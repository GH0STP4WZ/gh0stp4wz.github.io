- Music Packs -

This is a list of Custom Music Packs for Mario Kart Wii. These are collections of music files compressed into .BRSTM files.
These packs can usually be dropped into the My Stuff folder.


⚠ Users Beware ⚠

Some of these packs contain a custom revo_kart.brsar file, which is an archive of a lot of the commonly used sounds in the game. Replacing this file will not carry over existing sound effects.
